# angular-pipe-order-by
This can be used to order the array object.

### Usage

REVERSE: Boolean
CASE_SENSITIVE: Boolean
```html
<div *ngFor="let item of data | orderBy : REVERSE : CASE_SENSITIVE">
  <!--   content -->
</div>
```
